# DC-DC Buck-Boost Converter

## 📌 Description
This project simulates a DC-DC buck-boost converter using MATLAB/Simulink. The converter can either increase or decrease the input voltage depending on the duty cycle.

## ⚙️ Features
- Step-up and step-down voltage conversion
- PWM-based switching control
- Duty cycle variation
- Output voltage regulation

## 🛠️ Tools Used
- MATLAB
- Simulink

## ▶️ How to Run
1. Open the .slx file in MATLAB
2. Run the simulation
3. Observe input and output voltages in Scope
